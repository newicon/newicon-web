## [0.1.0] - 29-09-2017

### Added
- Component readme file
- Component Change log file
- Second tier of spacing variables - moved out of default theme
- Default theme colours - moved out of default theme

### Changed

e.g. changing @dark-green to @themeSecondary.

### Removed
